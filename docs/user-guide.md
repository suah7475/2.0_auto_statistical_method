# 통계 기법 추천 시스템 — 사용법 안내서

## 이 시스템은 뭔가요?

"어떤 통계 방법을 써야 하지?" 하고 고민될 때 사용하는 도구입니다.

간단한 질문 몇 개에 답하면, 내 연구에 맞는 통계 방법을 추천해 줍니다.
추천받은 방법에 대해 쉬운 설명도 보여주고, 바로 쓸 수 있는 R 코드도 만들어 줍니다.

재활 분야 연구를 처음 시작하는 분들을 위해 만들었습니다.

## 1단계: 질문에 답하기

화면에 질문이 하나씩 나타납니다. 내 연구에 맞는 답을 클릭하면 됩니다.
총 8개 질문이 있지만, 내 답에 따라 일부는 건너뛰기도 합니다.

### 질문 1. 결과 변수는 어떤 종류인가요?

내가 알아보고 싶은 "결과"가 숫자인지, 그룹(종류)인지 선택합니다.

| 선택지 | 이럴 때 선택하세요 | 예시 |
|--------|-------------------|------|
| **숫자** | 결과가 점수, 시간, 거리 같은 숫자일 때 | 통증 점수 7점, 보행 속도 1.2m/s, 악력 25kg |
| **그룹/종류** | 결과가 "이것 아니면 저것"일 때 | 치료 성공/실패, 정상/비정상, 경증/중증 |

> 쉽게 말하면: "결과를 자(ruler)로 잴 수 있으면 숫자, 분류할 수 있으면 그룹"

### 질문 2. 결과 그룹은 몇 가지이고, 순서가 있나요?

(질문 1에서 "그룹/종류"를 선택한 경우에만 나타납니다)

| 선택지 | 이럴 때 선택하세요 | 예시 |
|--------|-------------------|------|
| **2가지** | 결과가 둘 중 하나일 때 | 성공/실패, 정상/비정상, 가능/불가능 |
| **3가지 이상 (순서 없음)** | 결과가 3개 이상이고 순서가 없을 때 | 치료A/B/C, 좋아하는 과일(사과/바나나/오렌지) |
| **3가지 이상 (순서 있음)** | 결과가 3개 이상이고 순서가 있을 때 | 경증→중등증→중증, 1등급→2등급→3등급 |

> "순서 있음"과 "순서 없음"의 차이:
> - 경증→중등증→중증은 순서가 있어요 (경증이 가장 가벼움)
> - 치료A/B/C는 순서가 없어요 (어느 게 더 큰지 비교 불가)

### 질문 3. 원인/조건 변수가 몇 개인가요?

결과에 영향을 주는 변수(원인)가 몇 개인지 선택합니다.

| 선택지 | 이럴 때 선택하세요 | 예시 |
|--------|-------------------|------|
| **1개** | 하나의 조건만 비교할 때 | 치료군 vs 대조군만 비교 |
| **2개 이상** | 여러 조건을 동시에 볼 때 | 치료군 + 성별, 나이 + 치료횟수 + 중증도 |

> 쉽게 말하면: "비교하고 싶은 것이 1가지면 1개, 여러 가지면 2개 이상"

### 질문 4. 원인/조건 변수는 어떤 종류인가요?

원인 변수가 그룹(종류)인지 숫자인지 선택합니다.
두 가지가 섞여있으면 **둘 다 선택**할 수 있습니다.

| 선택지 | 이럴 때 선택하세요 | 예시 |
|--------|-------------------|------|
| **그룹** | 사람들을 나눠서 비교할 때 | 치료 종류(A/B), 성별(남/여), 그룹 분류 |
| **숫자** | 숫자로 예측하고 싶을 때 | 나이 45세, 치료 횟수 12회, BMI 24.5 |

둘 다 해당되면 두 개 모두 체크한 뒤 **"선택 완료"** 버튼을 누르세요.

### 질문 5. 비교할 그룹이 몇 개인가요?

(질문 4에서 "그룹"을 선택한 경우에만 나타납니다)

| 선택지 | 이럴 때 선택하세요 | 예시 |
|--------|-------------------|------|
| **2개 그룹** | 두 그룹을 비교할 때 | 실험군 vs 대조군, 남자 vs 여자 |
| **3개 이상** | 세 그룹 이상을 비교할 때 | 치료A vs 치료B vs 치료C |

### 질문 6. 같은 사람을 여러 번 측정했나요?

(질문 4에서 "그룹"을 선택한 경우에만 나타납니다)

| 선택지 | 이럴 때 선택하세요 | 예시 |
|--------|-------------------|------|
| **다른 사람들 비교** | 서로 다른 사람끼리 비교할 때 | 실험군 20명 vs 대조군 20명 (다른 사람) |
| **같은 사람 전-후 비교** | 같은 사람을 여러 번 측정했을 때 | 환자 20명의 치료 전 점수 vs 치료 후 점수 |

> 쉽게 말하면:
> - "다른 사람들": A반 학생 vs B반 학생 비교
> - "같은 사람 전-후": 같은 학생의 중간고사 vs 기말고사 비교

### 질문 7. 데이터가 정규분포인가요?

(결과가 "숫자"이고, 원인 변수에 "그룹"이 포함된 경우에만 나타납니다)

| 선택지 | 이럴 때 선택하세요 |
|--------|-------------------|
| **예 (정규분포)** | 히스토그램이 종 모양이거나, Shapiro-Wilk 검정 p값 > 0.05일 때 |
| **아니오 / 잘 모름** | 데이터가 한쪽으로 치우쳐 있거나, 잘 모르겠을 때 |

> 잘 모르겠으면 **"아니오 / 잘 모름"**을 선택하세요. 더 안전한 방법을 추천해 줍니다.

### 질문 8. 통제하고 싶은 다른 요인이 있나요?

(질문 3에서 "2개 이상"을 선택한 경우에만 나타납니다)

| 선택지 | 이럴 때 선택하세요 | 예시 |
|--------|-------------------|------|
| **없어요** | 다른 요인을 따로 빼고 싶지 않을 때 | |
| **있어요 (공변량)** | 특정 요인의 영향을 빼고 순수한 효과만 보고 싶을 때 | 나이 영향을 빼고 치료 효과만 보고 싶어요 |

### 이전 / 처음부터 버튼

- **← 이전**: 바로 앞 질문으로 돌아갑니다. 답을 바꿀 수 있어요.
- **처음부터**: 모든 답을 지우고 처음부터 다시 시작합니다.

### 분석 경로

예: "결과 변수가 숫자예요 → 비교할 변수가 1개예요 → 2개 그룹 비교 → 서로 다른 사람들이에요 → 정규분포예요"

### 추천된 통계 방법

세 가지 설명이 차례로 나옵니다:

1. **쉬운 설명**: 이 통계 방법이 뭔지 한 문장으로 설명
2. **비유**: 일상생활에 비유해서 이해하기 쉽게 설명
3. **재활 예시**: 재활 분야에서 실제로 어떻게 쓰는지 예시

**"자세한 설명 보기"**를 클릭하면 전문적인 설명, 가정 사항, 사용 조건 등을 볼 수 있습니다.

## 3단계: R 코드 만들기

추천 결과 아래에 **"R 코드 커스터마이저"**가 있습니다.
내 엑셀 데이터에 맞춰 바로 쓸 수 있는 R 코드를 만들어 줍니다.

### Step 1. 데이터 파일 정보 입력

1. **파일 경로** (선택사항): R에서 파일을 불러올 경로를 입력합니다
   - 예: `C:/Users/홍길동/Desktop/data.xlsx`
   - 입력하지 않으면 기본 경로가 사용됩니다

2. **파일 형식**: Excel(.xlsx) 또는 CSV(.csv)를 선택합니다

3. **컬럼(열 이름) 입력**: 엑셀의 첫 번째 줄(헤더)을 복사해서 붙여넣습니다
   - 쉼표로 구분: `환자ID, 성별, 나이, VAS_pre, VAS_post, 치료그룹`
   - 엑셀에서 직접 복사 붙여넣기도 됩니다 (탭으로 구분됨)

4. **"컬럼 감지"** 버튼을 클릭합니다

### Step 2. 컬럼 타입 확인

시스템이 각 컬럼의 데이터 타입을 자동으로 감지합니다.

| 타입 | 뜻 | 예시 |
|------|-----|------|
| 연속형 (숫자) | 숫자로 된 데이터 | 나이, 점수, 거리, 무게 |
| 범주형 (그룹) | 그룹으로 나뉘는 데이터 | 성별, 치료그룹, 진단명 |
| 순서형 (등급) | 순서가 있는 그룹 | 경증/중등증/중증, Likert 척도 |

자동 감지 결과가 맞는지 확인하고, 틀린 부분은 드롭다운에서 수정합니다.
확인이 끝나면 **"확인하고 다음으로"** 버튼을 클릭합니다.

### Step 3. 분석 변수 매핑

통계 분석에 필요한 변수를 내 컬럼에서 선택합니다.

예를 들어, 독립표본 t-검정이라면:
- **종속변수(결과)**: VAS_post (치료 후 점수)
- **그룹 변수**: 치료그룹 (실험군/대조군)

각 드롭다운에서 해당하는 컬럼을 선택한 뒤 **"R 코드 생성"** 버튼을 클릭합니다.

### 코드 사용하기

생성된 R 코드가 화면에 표시됩니다. 두 가지 방법으로 사용할 수 있습니다:

- **"코드 복사"** 버튼: 코드가 클립보드에 복사됩니다. R/RStudio에 붙여넣기(Ctrl+V)하면 됩니다.
- **".R 파일 다운로드"** 버튼: .R 파일로 저장됩니다. R/RStudio에서 열어서 실행하면 됩니다.

생성된 코드에는 다음이 모두 포함되어 있습니다:
1. 필요한 패키지 설치 및 불러오기
2. 엑셀/CSV 파일 읽기
3. 데이터 전처리
4. 기술통계 (평균, 표준편차 등)
5. 가정 검정 (정규성, 등분산성 등)
6. 본 분석 실행
7. 사후 검정 (필요한 경우)
8. 효과 크기
9. 그래프 그리기
10. 결과 해석 가이드 (한국어)

## 4단계: 논문 검색하기

결과 페이지 아래쪽에 **"관련 논문 검색 — PubMed"**가 있습니다.
추천된 통계 방법과 관련된 재활 분야 논문을 찾아줍니다.

### 사용 방법

1. **재활 분야 선택**: 드롭다운에서 내 분야를 선택합니다
   - 물리치료, 작업치료, 언어치료, 재활심리, 신경재활, 전체

2. **SCI-E급 저널만 표시**: 체크하면 SCI-E급 저널 논문만 보여줍니다
   - 기본값은 체크되어 있습니다
   - 결과가 적으면 체크를 해제해 보세요

3. **"논문 검색"** 버튼을 클릭합니다

### 검색 결과 보기

각 논문에는 다음 정보가 표시됩니다:
- **논문 제목** (클릭하면 PubMed 페이지로 이동)
- **저자** | **저널 이름** | **발행 연도**
- **SCI-E 배지**: SCI-E급 저널이면 배지가 표시됩니다

## 자주 묻는 질문

### Q. 정규분포인지 어떻게 알 수 있나요?

R에서 `shapiro.test(데이터)` 명령어로 확인할 수 있습니다.
p값이 0.05보다 크면 정규분포로 볼 수 있습니다.
잘 모르겠으면 "아니오"를 선택하세요. 더 안전한 비모수 검정을 추천해 줍니다.

### Q. 대응 표본(같은 사람 반복 측정)과 독립 표본(다른 사람 비교)이 헷갈려요.

- **같은 사람을 2번 이상 측정**했으면 → "같은 사람 전-후 비교" (대응 표본)
  - 예: 환자 20명의 치료 전 점수와 치료 후 점수 비교
- **서로 다른 사람**을 비교하면 → "다른 사람들 비교" (독립 표본)
  - 예: 실험군 20명과 대조군 20명 비교

### Q. 공변량(통제 변수)이 뭔가요?

내가 보고 싶은 효과 외에, 결과에 영향을 줄 수 있는 다른 변수입니다.

예: "물리치료 효과를 보고 싶은데, 환자 나이가 결과에 영향을 줄 수 있어요"
→ 이때 나이가 공변량입니다. 나이의 영향을 빼고 순수한 치료 효과만 볼 수 있습니다.

### Q. R을 처음 써보는데, 코드를 어떻게 실행하나요?

1. [R 다운로드](https://cran.r-project.org/) 페이지에서 R을 설치합니다
2. [RStudio 다운로드](https://posit.co/download/rstudio-desktop/) 페이지에서 RStudio를 설치합니다
3. RStudio를 열고, 왼쪽 콘솔 창에 코드를 붙여넣기합니다
4. Enter를 누르면 실행됩니다

처음 실행할 때 패키지 설치에 시간이 조금 걸릴 수 있습니다.

### Q. 엑셀 컬럼 이름이 한국어인데 괜찮나요?

네, 괜찮습니다. 이 시스템은 한국어 컬럼 이름을 자동으로 안전하게 처리합니다.
"환자번호", "치료그룹", "VAS점수" 같은 이름 그대로 사용하면 됩니다.

### Q. 추천 결과가 맞는지 어떻게 확신할 수 있나요?

이 시스템은 UCLA OARC의 통계 방법 선택 가이드라인을 기반으로 만들어졌습니다.
다만, 실제 연구에서는 지도교수나 통계 전문가의 검토를 받는 것을 권장합니다.

### Q. 다른 통계 방법으로 바꾸고 싶어요.

결과 화면에서 **"다른 방법도 있어요"** 섹션의 대안을 클릭하면 해당 방법으로 전환됩니다.
처음부터 다시 하고 싶으면 **"다른 분석 해보기"** 버튼을 누르세요.
