# 통계 기법 추천 시스템 — 사용법 안내서

## 이 시스템은 뭔가요?

"어떤 통계 방법을 써야 하지?" 하고 고민될 때 사용하는 도구입니다.

간단한 질문 몇 개에 답하거나, 목록에서 직접 통계 기법을 골라 설명을 확인할 수 있습니다.
추천받은 방법에 대해 쉬운 설명을 보여주고, 내 데이터에 맞는 R 코드를 자동으로 만들어주며,
브라우저에서 바로 실행까지 할 수 있습니다.

재활 분야 연구를 처음 시작하는 분들을 위해 만들었습니다.

## 방법 A: 단계별 질문으로 찾기

화면에 질문이 하나씩 나타납니다. 내 연구에 맞는 답을 클릭하면 됩니다.
총 8개 질문이 있지만, 내 답에 따라 일부는 건너뜁니다.

### 질문 1. 결과 변수는 어떤 종류인가요?

| 선택지 | 이럴 때 선택하세요 | 예시 |
|--------|-------------------|------|
| **숫자** | 결과가 점수, 시간, 거리 같은 숫자일 때 | 통증 점수 7점, 보행 속도 1.2m/s, 악력 25kg |
| **그룹/종류** | 결과가 "이것 아니면 저것"일 때 | 치료 성공/실패, 정상/비정상, 경증/중증 |

> 쉽게 말하면: "결과를 자(ruler)로 잴 수 있으면 숫자, 분류할 수 있으면 그룹"

### 질문 2. 결과 그룹은 몇 가지이고, 순서가 있나요?

(질문 1에서 "그룹/종류"를 선택한 경우에만 나타납니다)

| 선택지 | 이럴 때 선택하세요 | 예시 |
|--------|-------------------|------|
| **2가지** | 결과가 둘 중 하나일 때 | 성공/실패, 정상/비정상 |
| **3가지 이상 (순서 없음)** | 결과가 3개 이상이고 순서가 없을 때 | 치료A/B/C |
| **3가지 이상 (순서 있음)** | 결과가 3개 이상이고 순서가 있을 때 | 경증→중등증→중증 |

### 질문 3. 원인/조건 변수가 몇 개인가요?

| 선택지 | 이럴 때 선택하세요 | 예시 |
|--------|-------------------|------|
| **1개** | 하나의 조건만 비교할 때 | 치료군 vs 대조군만 비교 |
| **2개 이상** | 여러 조건을 동시에 볼 때 | 치료군 + 성별, 나이 + 치료횟수 |

### 질문 4. 원인/조건 변수는 어떤 종류인가요?

| 선택지 | 이럴 때 선택하세요 | 예시 |
|--------|-------------------|------|
| **그룹** | 사람들을 나눠서 비교할 때 | 치료 종류(A/B), 성별(남/여) |
| **숫자** | 숫자로 예측하고 싶을 때 | 나이 45세, 치료 횟수 12회 |

둘 다 해당되면 두 개 모두 체크한 뒤 **"선택 완료"** 버튼을 누르세요.

### 질문 5. 비교할 그룹이 몇 개인가요?

(질문 4에서 "그룹"을 선택한 경우에만 나타납니다)

| 선택지 | 이럴 때 선택하세요 | 예시 |
|--------|-------------------|------|
| **2개 그룹** | 두 그룹을 비교할 때 | 실험군 vs 대조군 |
| **3개 이상** | 세 그룹 이상을 비교할 때 | 치료A vs 치료B vs 치료C |

### 질문 6. 같은 사람을 여러 번 측정했나요?

(질문 4에서 "그룹"을 선택한 경우에만 나타납니다)

| 선택지 | 이럴 때 선택하세요 | 예시 |
|--------|-------------------|------|
| **다른 사람들 비교** | 서로 다른 사람끼리 비교할 때 | 실험군 20명 vs 대조군 20명 |
| **같은 사람 전-후 비교** | 같은 사람을 여러 번 측정했을 때 | 환자 20명의 치료 전 vs 치료 후 점수 |

### 질문 7. 데이터가 정규분포인가요?

(결과가 "숫자"이고 원인 변수에 "그룹"이 포함된 경우에만 나타납니다)

| 선택지 | 이럴 때 선택하세요 |
|--------|-------------------|
| **예 (정규분포)** | 히스토그램이 종 모양이거나, Shapiro-Wilk 검정 p값 > 0.05일 때 |
| **아니오 / 잘 모름** | 데이터가 한쪽으로 치우쳐 있거나, 잘 모르겠을 때 |

> 잘 모르겠으면 **"아니오 / 잘 모름"** 선택. 더 안전한 방법을 추천해 줍니다.

### 질문 8. 통제하고 싶은 다른 요인이 있나요?

(질문 3에서 "2개 이상"을 선택한 경우에만 나타납니다)

| 선택지 | 이럴 때 선택하세요 | 예시 |
|--------|-------------------|------|
| **없어요** | 다른 요인을 따로 빼고 싶지 않을 때 | |
| **있어요 (공변량)** | 특정 요인의 영향을 빼고 순수한 효과만 보고 싶을 때 | 나이 영향을 빼고 치료 효과만 보기 |

## 방법 B: 목록에서 직접 찾기

처음 화면에서 **"목록에서 직접 찾기"**를 선택하면 26가지 통계 기법이 카테고리별로 나열됩니다.

| 카테고리 | 포함된 기법 |
|----------|------------|
| 그룹 비교 (모수) | 일표본/독립/대응 t-검정, 일원/이원/반복/혼합 분산분석, ANCOVA |
| 그룹 비교 (비모수) | Mann-Whitney, Wilcoxon, Kruskal-Wallis, Friedman |
| 범주형 분석 | 카이제곱, Fisher 정확, McNemar |
| 상관 분석 | Pearson, Spearman, 점이연 상관 |
| 회귀 분석 | 단순/다중/더미 회귀, 이분형/다중/순서형 로지스틱 |
| 일반선형모형 | GLM (공변량 포함), GLM ANOVA |

## R 코드 만들기

추천 결과 아래 **"R 코드 커스터마이저"**에서 내 데이터에 맞는 R 코드를 만들 수 있습니다.
두 가지 모드를 선택할 수 있습니다.

| 모드 | 설명 |
|------|------|
| **R 코드 생성** | 코드를 복사해서 R/RStudio에 직접 붙여넣기 |
| **브라우저에서 실행** | 데이터 파일을 업로드하면 브라우저에서 바로 분석 실행 (서버 불필요) |

### R 코드 생성 모드

**Step 1. 데이터 파일 정보 입력**

1. **파일 경로** (선택사항): R에서 파일을 불러올 경로 입력
   - 예: `C:/Users/홍길동/Desktop/data.xlsx`
2. **파일 형식**: Excel(.xlsx) 또는 CSV(.csv) 선택
3. **컬럼(열 이름) 입력**: 엑셀 첫 번째 줄(헤더)을 복사해서 붙여넣기
   - 예: `환자ID, 성별, 나이, VAS_pre, VAS_post, 치료그룹`
4. **"컬럼 감지"** 버튼 클릭

**Step 2. 컬럼 타입 확인**

| 타입 | 뜻 | 예시 |
|------|-----|------|
| 연속형 (숫자) | 숫자로 된 데이터 | 나이, 점수, 거리 |
| 범주형 (그룹) | 그룹으로 나뉘는 데이터 | 성별, 치료그룹 |
| 순서형 (등급) | 순서가 있는 그룹 | 경증/중등증/중증 |

자동 감지 결과를 확인하고 틀린 부분은 수정한 뒤 **"확인하고 다음으로"** 클릭.

**Step 3. 분석 변수 매핑**

분석에 필요한 변수를 내 컬럼에서 선택합니다.

예(독립표본 t-검정):
- 종속변수: VAS_post
- 그룹 변수: 치료그룹

**"R 코드 생성"** 버튼 클릭 후:
- **"코드 복사"**: 클립보드에 복사 → R/RStudio에 붙여넣기
- **".R 파일 다운로드"**: .R 파일로 저장

생성된 코드에 포함된 항목:
1. 필요한 패키지 설치 및 불러오기
2. 데이터 파일 읽기
3. 데이터 전처리 (결측치 처리, 변수 형변환)
4. 기술통계 (평균, 표준편차 등)
5. 가정 검정 (정규성, 등분산성 등)
6. 본 분석 실행
7. 사후 검정 (필요한 경우)
8. 효과 크기
9. 그래프 그리기
10. 결과 해석 가이드 (한국어)
11. 엑셀 결과 내보내기

### 브라우저 실행 모드 (v2.0 신기능)

R을 설치하지 않아도 브라우저에서 바로 분석을 실행할 수 있습니다.

**준비사항**
- 분석할 데이터가 담긴 **CSV 파일** (Excel → 다른 이름으로 저장 → CSV)
- 인터넷 연결 (첫 실행 시 R 엔진과 패키지 다운로드 필요, 약 1-3분)

**사용 방법**

1. **"브라우저에서 실행"** 탭 선택
2. **CSV 파일 업로드**: "파일 선택" 버튼 클릭 후 CSV 파일 선택
3. **컬럼 타입 확인**: 각 컬럼의 타입(연속형/범주형) 확인 및 수정
4. **변수 매핑**: 분석에 필요한 변수를 내 컬럼에서 선택
   - 변수 선택란 옆에 **변수 유형 선택기**(연속형/범주형/순서형)가 표시됩니다
   - 변수를 선택하면 자동 감지된 유형이 미리 채워지며, 필요하면 직접 수정할 수 있습니다
   - 선택된 유형은 R 코드 실행 시 `as.numeric` / `as.factor` / `as.ordered` 변환으로 반영됩니다
5. **"분석 실행"** 버튼 클릭

**실행 과정**

| 단계 | 내용 | 소요 시간 |
|------|------|----------|
| R 엔진 초기화 | WebR(브라우저용 R) 로딩 | 10-30초 |
| 패키지 설치 | rstatix, effectsize 등 자동 설치 | 1-3분 (첫 실행만) |
| 분석 실행 | 데이터 로드 → 기술통계 → 검정 → 효과크기 | 10-30초 |

> 첫 실행 시 패키지 설치로 시간이 걸립니다. 같은 브라우저에서 재실행하면 캐시로 빠르게 시작됩니다.

**결과 확인**

분석이 완료되면 단계별로 결과가 표시됩니다:

- **데이터 불러오기**: 데이터 구조 확인
- **기술통계**: 그룹별 평균, 표준편차
- **가정 검정**: 정규성, 등분산성 검정
- **본 분석**: 검정 통계량, p값
- **효과크기**: Cohen's d, eta-squared 등
- **시각화**: 박스플롯, 산점도 등 그래프

> 오류가 발생한 단계는 `[WEBR_ERROR]` 로 표시됩니다. 해당 단계만 실패하고 나머지는 계속 실행됩니다.

> R 출력의 공백 정렬(열 구분)이 그대로 보존됩니다. 넓은 표는 **가로 스크롤**로 확인할 수 있습니다.

**지원 브라우저**

Chrome, Edge (최신 버전 권장). Firefox도 가능하지만 일부 기능이 제한될 수 있습니다.

## 논문 검색하기

결과 페이지 아래쪽에 **"관련 논문 검색 — PubMed"**가 있습니다.

**사용 방법**

1. **재활 분야 선택**: 드롭다운에서 내 분야 선택
   - 물리치료, 작업치료, 언어치료, 재활심리, 신경재활, 전체
2. **SCI-E급 저널만 표시**: 체크하면 SCI-E급 저널 논문만 표시 (기본: 체크됨)
3. **"논문 검색"** 버튼 클릭

**검색 결과**

각 논문에 표시되는 정보:
- 논문 제목 (클릭 시 PubMed 페이지로 이동)
- 저자 | 저널 이름 | 발행 연도
- SCI-E 배지: SCI-E급 저널이면 표시

## 자주 묻는 질문

### Q. 정규분포인지 어떻게 알 수 있나요?

R에서 `shapiro.test(데이터)` 명령어로 확인할 수 있습니다.
p값이 0.05보다 크면 정규분포로 볼 수 있습니다.
잘 모르겠으면 "아니오"를 선택하세요. 더 안전한 비모수 검정을 추천해 줍니다.

### Q. 대응 표본과 독립 표본이 헷갈려요.

- **같은 사람을 2번 이상 측정**했으면 → "같은 사람 전-후 비교" (대응 표본)
  - 예: 환자 20명의 치료 전 점수와 치료 후 점수 비교
- **서로 다른 사람**을 비교하면 → "다른 사람들 비교" (독립 표본)
  - 예: 실험군 20명과 대조군 20명 비교

### Q. 공변량(통제 변수)이 뭔가요?

내가 보고 싶은 효과 외에 결과에 영향을 줄 수 있는 다른 변수입니다.

예: "물리치료 효과를 보고 싶은데, 환자 나이가 결과에 영향을 줄 수 있어요"
→ 이때 나이가 공변량입니다. 나이의 영향을 빼고 순수한 치료 효과만 볼 수 있습니다.

### Q. R을 처음 써보는데, 코드를 어떻게 실행하나요?

브라우저 실행 모드를 사용하면 R 설치 없이 바로 분석할 수 있습니다.

R/RStudio 사용을 원한다면:
1. [R 다운로드](https://cran.r-project.org/)에서 R 설치
2. [RStudio 다운로드](https://posit.co/download/rstudio-desktop/)에서 RStudio 설치
3. RStudio에서 코드를 붙여넣고 실행

처음 실행 시 패키지 설치에 시간이 걸릴 수 있습니다.

### Q. 엑셀 컬럼 이름이 한국어인데 괜찮나요?

네, 괜찮습니다. 한국어 컬럼 이름을 자동으로 안전하게 처리합니다.
"환자번호", "치료그룹", "VAS점수" 같은 이름 그대로 사용하면 됩니다.

### Q. 브라우저 실행 시 데이터가 외부로 전송되나요?

아니오. 데이터는 브라우저 안에서만 처리됩니다. 서버로 전송되지 않습니다.
WebR은 R 엔진 자체를 브라우저 안에서 실행하는 기술입니다.

### Q. 브라우저 실행 모드에서 CSV 파일이 필요한 이유는?

브라우저 실행 모드는 R 코드를 직접 실행하므로 실제 데이터 파일이 필요합니다.
Excel 파일(.xlsx)은 현재 지원하지 않습니다. Excel에서 "다른 이름으로 저장 → CSV"로 변환해 주세요.

### Q. 추천 결과가 맞는지 어떻게 확인하나요?

이 시스템은 UCLA OARC의 통계 방법 선택 가이드라인을 기반으로 만들어졌습니다.
실제 연구에서는 지도교수나 통계 전문가의 검토를 받는 것을 권장합니다.
